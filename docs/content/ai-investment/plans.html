üìù Plan Configuration Fields

Basic Information

Plan Name
Text (Required)

The internal name for the investment plan. Used for identification in the admin panel.

Title
Text (Required)

The public-facing title displayed to users. Should be descriptive and attractive.

Description
Text (Optional)

Detailed description of the investment plan, its strategy, and expected outcomes.

Image
URL (Optional)

URL to an image representing the investment plan. Falls back to placeholder if not provided.

Financial Parameters

Invested Amount
Number (Required)

Total amount currently invested in this plan across all users.

Profit Percentage
Number (Required)

Base profit percentage for calculations. Used as reference for profit generation.

Minimum Profit
Number (Required)

Lowest possible profit percentage that can be generated.

Maximum Profit
Number (Required)

Highest possible profit percentage that can be generated.

Default Profit
Number (Required)

Standard profit value used when specific calculations are not applied.

Investment Limits

Minimum Amount
Number (Required)

Smallest investment amount allowed for this plan.

Maximum Amount
Number (Required)

Largest investment amount allowed for this plan.

Plan Settings

Default Result
Select (Required)

Default outcome for investments in this plan:

WIN LOSS DRAW
Status
Boolean

Whether the plan is active and available for new investments.

Trending
Boolean

Mark as trending to feature prominently in user interfaces.

Durations
Array

List of available investment duration options for this plan.

üí° Best Practices

Profit Configuration

  • ‚Ä¢ Realistic Ranges - Set achievable profit ranges based on market conditions
  • ‚Ä¢ Risk Assessment - Balance higher profits with appropriate risk levels
  • ‚Ä¢ Market Research - Base profit percentages on actual market performance

Investment Limits

  • ‚Ä¢ Minimum Amounts - Set reasonable minimums to ensure plan viability
  • ‚Ä¢ Maximum Limits - Prevent over-concentration in single plans
  • ‚Ä¢ Tier Structure - Consider multiple plans for different investment levels

Plan Lifecycle

  • ‚Ä¢ Testing Phase - Start with limited amounts and short durations
  • ‚Ä¢ Gradual Scaling - Increase limits based on performance
  • ‚Ä¢ Regular Review - Monitor and adjust parameters based on results

‚ö†Ô∏è Important Considerations

  • ‚Ä¢ Active Investments - Plans with active investments cannot be deleted
  • ‚Ä¢ Parameter Changes - Modifications only affect new investments, not existing ones
  • ‚Ä¢ Status Changes - Disabling a plan stops new investments but doesn't affect active ones
  • ‚Ä¢ Duration Dependencies - Ensure associated durations exist before assigning them
