FAQ Categories

🎯 Category Management Overview

Categories help organize your FAQ content into logical groups. Each FAQ question belongs to a category, making it easier for users to find relevant information.

🏗️ Category System

How Categories Work

Simple Text Labels

Categories are simple text strings

One Category per FAQ

Each FAQ belongs to one category

Auto-Created

Categories are created when used in FAQs

➕ Using Categories

Assigning Categories to FAQs

1

Create or Edit FAQ

Go to FAQ → Manage and create/edit an FAQ

2

Enter Category Name

Type the category name in the category field

3

Save FAQ

The category will be automatically created if it doesn't exist

Naming Guidelines

Be Descriptive

Use clear, descriptive category names

Keep Consistent

Use the same name format across FAQs

Avoid Duplicates

Check existing categories before creating new ones

Best Practices

Logical Grouping

Group related questions together

User-Friendly

Use terms users understand

Not Too Many

Keep categories manageable (5-15 total)

👀 Viewing Categories

You can view all existing categories and their FAQ counts in the admin panel.

Category List

Access FAQ → Categories to see:

  • • All category names
  • • Number of FAQs per category
  • • Quick navigation to FAQs

Category Management

Categories are managed through FAQs:

  • • Created when assigning to FAQs
  • • Updated by changing FAQ categories
  • • Removed when no FAQs use them

💡 Best Practices

Organization

  • • Plan your category structure
  • • Use consistent naming conventions
  • • Group similar topics together
  • • Consider user mental models
  • • Review and refine regularly

Maintenance

  • • Check for typos in category names
  • • Merge similar categories
  • • Update categories as business evolves
  • • Monitor category usage
  • • Keep categories relevant
