Import Existing Templates

Import templates from external sources or backup files:

Import Process

  • • Access import function via API endpoint
  • • Upload template data in JSON format
  • • Validate template structure
  • • Save imported template to database
  • • Confirm successful import

Import Requirements

  • • Template name must be provided
  • • Content field is required
  • • Design information must be included
  • • Valid JSON format required
  • • No duplicate names allowed
