1

Navigate to Token Management

Go to Admin → Extensions → Ecosystem → Tokens and enable the tokens you plan to use for your blockchain network.

2

Configure Token Settings

Click the 3 dots next to a token and select Edit to configure:

Fee Configuration

  • • Minimum Fee: Set a minimum fee (e.g., $5) to cover blockchain transaction costs when percentage fees are too low
  • • Percentage Fee: Standard withdrawal fee based on a percentage of the withdrawn amount

Transaction Limits

  • • Deposit Limits: Minimum/maximum amount users can deposit for this token
  • • Withdraw Limits: Minimum/maximum amount users can withdraw for this token

Token Types Overview

• Native: The main coin of a blockchain used to pay network fees
• PERMIT: Advanced tokens allowing fee delegation - users don't need native coins for each transaction
• NO PERMIT: Tokens requiring custodial wallets - the master wallet pays gas fees for withdrawals
