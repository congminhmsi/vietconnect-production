🎯 Transaction Overview

ICO transactions represent individual token purchases within your ICO offers. Each transaction contains basic information about the purchase amount, price, payment details, and status tracking.

🔍 Accessing Transactions

Navigate to Transaction Management

Access the ICO transaction management section:

Admin Panel → ICO → Transactions

This provides a view of all ICO transactions across all offers with basic filtering and viewing capabilities.

💡 Best Practices

Transaction Management

  • • Regular monitoring of pending transactions
  • • Prompt verification and status updates
  • • Clear admin notes for audit trail
  • • Verify wallet addresses before release
  • • Document all status changes

Record Keeping

  • • Maintain detailed transaction logs
  • • Store release URLs for reference
  • • Track relationships between users and offers
  • • Regular backup of transaction data
  • • Monitor for duplicate transactions
