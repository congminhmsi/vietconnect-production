➕ Creating Forex Accounts

Access Account Management

Navigate to Admin Panel → Forex → Accounts and click "Create Account"

Required Fields

Account balance amount

DEMO or LIVE account type

Optional Fields

Associated user identifier

Trading account identifier

Account password/secret

Broker name or identifier

MT version (4 or 5)

Account leverage ratio

Active/inactive status
