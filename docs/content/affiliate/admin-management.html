📜 Condition Management

Configure reward conditions and rules at /admin/affiliate/condition to define how affiliates earn commissions and bonuses.

⚙️ Condition Configuration

🎯 Reward Types

Signup Rewards
Commission for successful user registrations
Deposit Rewards
Percentage or fixed amount on user deposits
Trading Rewards
Ongoing commission from trading activities

🔧 Configuration Options

  • Commission Rates: Set percentage or fixed amounts
  • Tier Structures: Multi-level commission hierarchies
  • Minimum Thresholds: Requirements for reward eligibility
  • Time Limits: Expiration dates and validity periods
  • User Conditions: KYC, activity, or deposit requirements
  • Geographic Rules: Location-based reward variations

🎛️ Advanced Rule Engine

Create sophisticated reward conditions using the built-in rule engine:

Conditional Logic
  • IF/THEN/ELSE rule structures
  • Multiple condition combinations
  • Date and time-based triggers
  • User behavior pattern matching
Performance Bonuses
  • Milestone achievement rewards
  • Volume-based bonus tiers
  • Time-limited promotional rates
  • Competition and leaderboard prizes

⚙️ System Settings

Configure global affiliate system settings at /admin/affiliate/settings to control system behavior and policies.

🔧 Core Settings

💰 Financial Configuration

  • Default Commission Rate: System-wide base commission percentage
  • Minimum Payout Amount: Threshold for commission withdrawals
  • Maximum Payout Limit: Cap on individual payout amounts
  • Payout Schedule: Frequency of commission payments
  • Currency Settings: Supported currencies and conversion rates

🛡️ Security & Compliance

  • KYC Requirements: Verification needed for affiliates
  • Fraud Detection: Automated monitoring thresholds
  • Geographic Restrictions: Country-based access controls
  • Age Verification: Minimum age requirements
  • Terms Acceptance: Required legal agreement acceptance

🎯 Operational Settings

🔗 Tracking Configuration

  • Cookie Duration: Referral tracking cookie lifetime
  • Attribution Model: First-click or last-click attribution
  • Cross-Device Tracking: Multi-device user recognition
  • Link Expiration: Referral link validity periods

📧 Communication Settings

  • Email Notifications: Automated email triggers
  • Welcome Messages: New affiliate onboarding
  • Performance Reports: Regular activity summaries
  • Payment Notifications: Payout confirmation emails

💡 Admin Best Practices

🚀 Program Optimization

  • Regular Performance Reviews: Weekly analysis of key metrics and trends
  • Competitive Benchmarking: Monitor industry standards and adjust rates
  • A/B Testing: Test different reward structures and conditions
  • Feedback Integration: Regularly survey affiliates for improvement ideas
  • Technology Updates: Keep tracking and analytics tools current

  • Fraud Monitoring: Implement automated detection systems
  • Regular Audits: Periodic review of affiliate activities and payouts
  • Legal Compliance: Stay updated with affiliate marketing regulations
  • Financial Controls: Set appropriate limits and approval workflows
  • Documentation: Maintain clear terms of service and guidelines
